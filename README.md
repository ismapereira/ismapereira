### 🌟 Sobre Mim

Sou um desenvolvedor fullstack focado em construir soluções robustas, rápidas e visualmente refinadas. Atuo no suporte técnico de um **provedor de internet**, o que me garante uma base sólida em infraestrutura, redes, tráfego de dados e resolução ágil de problemas.

### 📂 Projetos em Destaque

#### 💎 **Sistema Gestart (SaaS de Gestão Empresarial)**
Um ecossistema comercial robusto construído sob medida para a gestão e otimização de micro e pequenas empresas de ponta a ponta.
* **Arquitetura Pura:** Desenvolvido em **PHP 8.1+ MVC** orientado a objetos, sem sobrecarga de frameworks pesados, rodando com extrema performance e isolamento de banco via **Multi-Tenancy**.
* **Frontend Ultramoderno:** Interface premium responsiva com **Tailwind CSS v4** e comportamento ultra-reativo via **Alpine.js** (sem a necessidade de pesados scripts SPA).
* **Recursos Avançados:** 
  * Geração dinâmica de orçamentos e relatórios financeiros em **PDF (White-Label)** via servidor.
  * Dashboards interativas de inteligência de negócios com **Chart.js** e agenda de compromissos integrada via **FullCalendar**.
  * Blindagem robusta de segurança contra CSRF, XSS (Sanitização), SQL Injection (PDO) e Device Session Lock.
  * Suporte a PWA nativo (Progressive Web App) instalável.

#### 🌳 **Green Air (Georreferenciamento Urbano)**
Uma plataforma colaborativa open-source voltada para o mapeamento, identificação e monitoramento de árvores urbanas, disponibilizando dados abertos para uso acadêmico e institucional.
* **Geolocalização Avançada:** Mapa interativo em tempo real alimentado por **Leaflet 1.9** com sistema de **MarkerCluster** de alta performance para renderizar múltiplos marcadores geograficamente.
* **Visualização de Dados:** Dashboard pública com análise estatística em gráficos dinâmicos de distribuição de espécies com **Chart.js 4**.
* **Arquitetura Robusta:** Back-end estruturado em **PHP 8.x MVC puro** com autoload PSR-4 (Composer), utilizando **Repository Pattern** para isolamento de queries SQL e **Service Layer** para desacoplamento de lógica de negócio.
* **Vite & Tailwind CSS v4:** Fluxo frontend ágil com build automatizado de CSS e JS nativos.
